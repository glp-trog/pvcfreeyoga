# pvcfreeyoga

Static site for pvcfreeyoga.com (mats-first, PVC-free positioning).

## Publish (GitHub Pages)
- Repo Settings → Pages → Deploy from branch `main` / folder `/docs`
- Custom domain: `pvcfreeyoga.com`
- Add `docs/CNAME` with the domain (once pointed)

## Local preview
Open `docs/index.html` in your browser.
